## git 설정 확인하기
git config --list

## 기본 브랜치명 수정하기
git init과 동시에 만들어지는 default 브랜치 이름이 master로 되어 있음 이를 main으로 바꾸기

git config --global init.defaultbranch main

## 명령어에 대한 설명을 확인하기
- git help <동사>
- git 동사 --help
- man git-동사

git help config
git config --help
내 컴퓨터안에 파일을 염

git <동사> -h
git config -h
